  • 拿到任何新数据，第一件事就是运行 df.head() 和 df.info()。
  • 如果 info() 显示某列的非空值 (Non-Null Count) 少于总行数，说明有缺失值。
  • 如果日期列的类型是 object (字符串) 而不是 datetime，后续时间分析会报错，需要转换。

处理缺失值

我们有三种策略：删除、填充、忽略。
策略 A：关键信息缺失（如“单价”为空），无法计算销售额，直接删除该行。
策略 B：非关键信息缺失（如“省份”为空），可以填充为“未知”。
策略 C：数值缺失（如“销量”），可以填充为 0 或平均值。
